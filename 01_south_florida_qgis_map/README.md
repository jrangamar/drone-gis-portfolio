# Arch Creek Park Site Context Map

## Overview

This project uses QGIS to create a site-context map for Arch Creek Park in North Miami Beach, Florida.

The map shows:

- Park boundary
- 250 m buffer
- Nearby streets
- OpenStreetMap basemap
- Map layout elements (legend, north arrow, scale bar)

The project was completed as an introductory GIS exercise focused on spatial analysis and cartographic design.

---

## Skills Demonstrated

- Loading and inspecting GIS datasets
- Working with coordinate reference systems (CRS)
- Reprojecting layers
- Creating distance-based buffers
- Clipping vector data
- Layer styling and symbology
- Map layout creation
- Exporting map products

---

## Workflow

1. Load Miami-Dade park and street datasets.
2. Select Arch Creek Park.
3. Export the selected park boundary.
4. Reproject layers into UTM Zone 17N.
5. Create a 250 m buffer around the park.
6. Clip streets to the buffer area.
7. Style layers for readability.
8. Design and export the final map layout.
