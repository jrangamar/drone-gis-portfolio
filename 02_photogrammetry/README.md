# Photogrammetry

This section documents drone-photogrammetry workflows using WebODM, Docker, and QGIS.

Objectives:

- Process aerial imagery into geospatial products.
- Explore terrain visualization techniques.
- Build practical GIS and remote sensing skills.
- Create portfolio-ready case studies.

## Software

- WebODM
- Docker Desktop
- QGIS

## Workflow

Drone Imagery
→ WebODM Processing
→ Orthophoto
→ DSM
→ Point Cloud
→ QGIS Analysis
→ Terrain Products

## Projects

### Sand Key

Introductory learning project.

Products:

- Orthophoto
- DSM
- Hillshade
- Contours

Focus:

- Learning the workflow.
- Validating software setup.
- Understanding photogrammetry outputs.
